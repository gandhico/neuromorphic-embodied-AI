# Neuromorphic Embodied AI

This repository contains research and development focused on Neuromorphic Embodied AI, organized into the following core areas:

* **Robotics Simulations** Simulation platforms used to integrate neuromorphic sensing and control for embodied AI. This includes dedicated environments for ROS 2, Gazebo, NVIDIA Isaac Sim, MuJoCo, Basilisk, Quanser, and X-Plane.
* **Neuromorphic Sensing & Control Closed-Loop Algorithms** Core sensing & control algorithms exploring the spectrum of autonomy. Features conventional, hybrid, and fully neuromorphic closed-loop implementations to benchmark different sensing & control strategies.
* **Hardware-In-The-Loop (HITL)/Real-Time Testing** Deployment pipelines that transition digital twins into physical applications. Includes robust, user-friendly interfaces for real-time testing and finalized project execution.
